# 회원 관리 앱 구조 및 기본 기능

이 문서는 현재 회원 관리 앱의 **구조(파일/모듈)**와 **기본 기능**을 빠르게 이해할 수 있도록 정리한 안내서입니다.

## 1) 프로젝트 구조

```
cm-login/
├─ index.html              # 앱의 기본 화면/레이아웃
├─ css/                    # 화면 스타일
│  ├─ base.css             # 공통 스타일
│  ├─ login.css            # 로그인/권한 UI
│  ├─ member.css           # 회원 목록/상세 UI
│  ├─ schedule.css         # 스케줄 UI
│  ├─ calendar.css         # 캘린더 UI
│  ├─ coach.css            # 코치 UI
│  └─ responsive.css       # 반응형 스타일
├─ js/                     # 기능 스크립트
│  ├─ base.js              # Firebase 초기화/전역 상태
│  ├─ login.js             # 로그인/권한 처리
│  ├─ member.js            # 회원 목록/정렬/검색
│  ├─ form.js              # 회원 등록/수정 폼
│  ├─ schedule.js          # 스케줄 렌더링
│  ├─ calendar.js          # 캘린더 로직
│  ├─ coach.js             # 코치 버튼/필터
│  ├─ settings.js          # 설정/관리자 관리
│  └─ modal.js             # 알림/확인 모달
├─ etc/
│  └─ manifest.json        # PWA 매니페스트
└─ sw.js                   # 서비스 워커 (오프라인 캐시)
```

## 2) 기본 기능

### ✅ 로그인/권한
- 관리자/부관리자/게스트 권한으로 UI를 제어합니다.
- Firebase Auth를 사용해 로그인 상태를 유지합니다.

### ✅ 회원 관리
- 회원 등록/수정/삭제 및 목록 표시
- 이름/전화번호 검색
- 등록일순/이름순/코치별 정렬

### ✅ 레슨/스케줄
- 회원별 레슨 목표/현재 횟수 관리
- 주간 스케줄 표시
- 캘린더 기반 레슨 관리 UI

### ✅ 회비/입금
- 월회비 입력 및 프리셋 제공
- 입금 내역 추가/표시

### ✅ 설정
- 구장명/회비 프리셋/코치 목록 관리
- 관리자/부관리자 계정 관리
- 계좌 정보 설정

# 파일별 주요 기능을 정리한 상세 설명

- **Firebase Realtime Database**에 `members`, `settings`, `admins` 데이터를 저장합니다.
- 앱 로드 시 `base.js`에서 데이터 로드 → 화면 렌더링.
- 폼/버튼 이벤트로 데이터 변경 → Firebase에 저장 → 변경 감지 후 UI 갱신.

📄 루트 파일
README.md: 프로젝트 구조와 핵심 기능(로그인/회원관리/스케줄/설정 등)을 요약한 안내서입니다.

index.html: 앱의 전체 레이아웃을 정의합니다. 헤더(구장명/회원수/설정/로그아웃), 검색/정렬/탭(회원 목록·스케줄), 회원 등록 폼(코치·레슨·회비·스케줄·성별/생년/부수 등), 모달 영역을 포함합니다.

sw.js: 서비스 워커로 오프라인 캐시를 관리합니다. 앱 리소스 캐싱, 네트워크 우선 전략, 캐시 업데이트 메시지 처리 등을 담당합니다.

수정된파일.txt: 날짜별 수정 이력과 변경된 파일 목록을 기록한 메모입니다.

📁 etc/
etc/manifest.json: PWA 매니페스트로 앱 이름/아이콘/시작 URL/테마 색상 등을 정의합니다.

📁 js/ (기능 스크립트)

js/base.js: 서비스 워커 등록, Firebase 초기화, 전역 상태(회원/설정), 데이터 로드/동기화, 저장 로직을 담당합니다.

js/login.js: 로그인/권한 시스템(관리자·부관리자·게스트), 내부 이메일 변환, Auth 상태 감지, 로그인 처리, 권한 UI 업데이트를 수행합니다.

js/member.js: 회원 검색/정렬/목록 렌더링, 코치별 그룹 표시, 회원 카드 UI 생성 및 버튼 권한 제어를 담당합니다.

js/form.js: 회원 등록/수정 폼 처리, 성별/수상경력/레슨 횟수/스케줄 검증, 신규 회원 생성 로직을 담당합니다.

js/calendar.js: 레슨 관리 캘린더 토글, 회원 레슨 문자(SMS) 발송 보조, 레슨 알림 모달과 알림음 재생 등 레슨 관련 유틸을 제공합니다.

js/schedule.js: 회원별 스케줄 입력 UI 렌더링, 스케줄 추가/삭제/저장/초기화 기능을 제공합니다.

js/settings.js: 설정 모달(구장명, 회비 프리셋, 코치, 계좌), 관리자/부관리자 목록 로드 및 계정 관리 기능을 담당합니다.

js/camera.js: 카메라 접근 및 촬영, 전/후면 전환, 사진 업로드/크롭/미리보기/삭제를 처리합니다.

js/modal.js: 레슨 초기화/삭제 확인 모달, 알림 모달, 커스텀 confirm 모달 표시를 처리합니다.

js/coach.js: 코치 버튼 렌더링/선택, 코치별 회원 수 집계, 코치별 섹션 접기/펼치기를 처리합니다.

📁 css/ (스타일)

css/base.css: 앱 전반 레이아웃/폰트/폼/버튼 등 공통 스타일을 정의합니다.

css/login.css: 로그인 섹션/모달/버튼 및 로그아웃 아이콘 스타일을 담당합니다.

css/member.css: 회원 카드, 연락처/회비 표시, 스케줄 배지, 상세 모달 스타일을 정의합니다.

css/camera.css: 사진 촬영/업로드 UI(프리뷰, 캡처 버튼, 모달)를 스타일링합니다.

css/calendar.css: 레슨 캘린더 UI(그리드, 날짜 상태, 알림 모달 등) 스타일을 정의합니다.

css/coach.css: 코치 버튼 및 코치별/요일별 섹션 UI 스타일을 담당합니다.

css/responsive.css: 모바일 화면 대응 레이아웃(폼/카드/캘린더/카메라 등) 반응형 스타일을 제공합니다.

css/schedule.css: 스케줄 카드와 추가/삭제 버튼 애니메이션 등 스케줄 영역 스타일을 담당합니다.
